=======

Java Unit Test Framework

### 功能

- Use H2 to mock db
- Use mockito to mock interface 
- Utils to mock get/set/construct/tostring

### Quick Start



### Maven

    <dependency>
        <groupId>com.github.knightliao.test</groupId>
        <artifactId>jutf-base</artifactId>
        <version>1.0.1-SNAPSHOT</version>
    </dependency>
